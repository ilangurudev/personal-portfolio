# Personal Portfolio

This project represents a dual-personality digital space. It separates my professional identity as an AI engineer from my artistic side as a photographer.

## Key Features

- **Dual Layout System**: Distinct visual themes for "Professional" (terminal/dark) and "Photography" (editorial/light).
- **AI-Assisted Workflow**: Leveraging LLM agents for content generation and code assistance.
- **Performance**: Static site generation with Astro for optimal load times.

## Tech Stack

- **Framework**: Astro 5.0
- **Styling**: Tailwind CSS
- **Content**: Markdown & MDX
